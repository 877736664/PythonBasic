# 06-元组 - 学习要点

## 核心知识点

### 1. 什么是元组
- 元组是**有序不可变**容器，一旦创建不能修改（增删改都不行）
- 用圆括号 `()` 表示，元素用逗号分隔
```python
empty = ()              # 空元组
t1 = (1, 2, 3)          # 整数元组
t2 = ("a", "b", "c")    # 字符串元组
t3 = (1, "hello", True) # 混合类型
# 重要：单元素元组必须加逗号！
single = (5,)            # 这才是元组
wrong = (5)              # 这只是一个整数，不是元组
```

### 2. 元组的基本操作
因为不可变，所以操作很简单：
- 索引访问：和列表一样 `t[0]`, `t[-1]`
- 切片：和列表一样 `t[1:3]`
- 遍历：`for item in t:`
- 长度：`len(t)`
- 拼接：`t1 + t2`
- 重复：`t * 3`

### 3. 元组常用场景
- 函数返回多个值就是返回元组
- 保护数据不被修改
- 解包：`a, b = (1, 2)` 或 `a, b = 1, 2`
