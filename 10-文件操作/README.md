# 10-文件操作 - 学习要点

## 核心知识点

### 1. 打开和关闭文件
```python
# 打开文件，推荐用with语句（自动关闭）
with open("filename.txt", "r", encoding="utf-8") as f:
    content = f.read()  # 读取全部内容

# mode:
# 'r' - 只读（默认）
# 'w' - 写入，覆盖原有内容
# 'a' - 追加，在末尾添加
# 'r+' - 读写
```

### 2. 读取文件
| 方法 | 作用 |
|------|------|
| `f.read()` | 读取全部内容，返回字符串 |
| `f.readline()` | 读取一行 |
| `f.readlines()` | 读取所有行，返回每行组成的列表 |
| `for line in f:` | 逐行迭代，内存友好 |

### 3. 写入文件
| 方法 | 作用 |
|------|------|
| `f.write(s)` | 写入字符串，返回写入字符数 |
| `f.writelines(lst)` | 写入多行列表 |
