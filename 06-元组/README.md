# 06-元组 - 学习要点

## 核心知识点

### 1. 什么是元组
- 元组是**有序不可变**容器，一旦创建不能修改（增删改都不行）
- 用圆括号 `()` 表示，元素用逗号分隔
```python
empty = ()              # 空元组
t1 = (1, 2, 3)          # 整数元组
t2 = ("a", "b", "c")    # 字符串元组
t3 = (1, "hello", True) # 混合类型
# 重要：单元素元组必须加逗号！
single = (5,)            # 这才是元组
wrong = (5)              # 这只是一个整数，不是元组
```

### 2. 元组和列表互相转换
```python
# 列表转元组
lst = [1, 2, 3]
t = tuple(lst)  # (1, 2, 3)

# 元组转列表
t = (1, 2, 3)
lst = list(t)  # [1, 2, 3]
```

### 3. 元组的基本操作
因为不可变，所以操作很简单：
- 索引访问：和列表一样 `t[0]`, `t[-1]`
- 切片：和列表一样 `t[start:end:step]`
- 遍历：`for item in t:`
- 长度：`len(t)`
- 拼接：`t1 + t2` → 产生新元组，不修改原元组
- 重复：`t * 3`
- 统计次数：`t.count(x)` → 返回x出现次数，和列表一样
- 判断包含：`x in t` → 返回True/False

### 4. 元组解包（Unpacking）
```python
# 基本解包
t = (10, 20)
a, b = t
# a = 10, b = 20

# *解包：收集中间元素
a, *b, c = (1, 2, 3, 4, 5)
# a = 1, b = [2, 3, 4], c = 5

# 交换变量（Python最简洁的写法）
a = 10
b = 20
a, b = b, a  # 交换完成，本质就是元组解包
```

### 5. 元组常用场景
- 函数返回多个值就是返回元组
```python
def divide(a, b):
    q = a // b
    r = a % b
    return q, r  # 返回一个元组

商, 余数 = divide(10, 3)
```
- 保护数据不被修改（作为常量）
- 字典的键可以用元组，不能用列表
