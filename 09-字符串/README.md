# 09-字符串 - 学习要点

## 核心知识点

### 1. 字符串基本概念
- 字符串是**不可变**的字符序列
- 定义：单引号 `'` 或双引号 `"` 或三引号 `'''`/`"""`
```python
s1 = 'hello'
s2 = "world"
s3 = """多行
字符串"""
```

### 2. 基本操作
- 索引：`s[0]`, `s[-1]` 和列表一样
- 切片：`s[start:end:step]` 和列表一样
- 长度：`len(s)`
- 拼接：`s1 + s2`
- 重复：`s * 3`
- 判断包含：`'a' in s`

### 3. 常用方法

**查找替换：**
| 方法 | 作用 |
|------|------|
| `s.find(sub)` | 返回子串第一次出现的索引，找不到返回-1 |
| `s.count(sub)` | 统计子串出现次数 |
| `s.replace(old, new)` | 替换所有匹配，返回新字符串 |

**分割连接：**
| 方法 | 作用 |
|------|------|
| `s.split(sep)` | 按分隔符分割成列表 |
| `sep.join(lst)` | 把列表元素用分隔符连接成字符串 |

**大小写转换：**
| 方法 | 作用 |
|------|------|
| `s.upper()` | 转大写 |
| `s.lower()` | 转小写 |
| `s.title()` | 每个单词首字母大写 |
| `s.strip()` / `lstrip()` / `rstrip()` | 去除两端/左端/右端空白 |

**判断：**
| 方法 | 作用 |
|------|------|
| `s.isdigit()` | 判断是否全是数字 |
| `s.startswith(prefix)` | 判断是否以prefix开头 |
| `s.endswith(suffix)` | 判断是否以suffix结尾 |
