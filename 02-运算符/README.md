# 02-运算符 - 学习要点

## 核心知识点

### 1. 算术运算符
| 运算符 | 说明 | 示例 | 结果 |
|--------|------|------|------|
| `+` | 加法 | `10 + 3` | `13` |
| `-` | 减法 | `10 - 3` | `7` |
| `*` | 乘法 | `10 * 3` | `30` |
| `/` | 除法（浮点数） | `10 / 3` | `3.333...` |
| `//` | 整除（向下取整） | `10 // 3` | `3` |
| `%` | 取余（取模） | `10 % 3` | `1` |
| `**` | 幂运算 | `10 ** 3` | `1000` |

- ⚠️ 注意：`/` 永远返回 float，`//` 是整除

### 2. 比较运算符
| 运算符 | 说明 |
|--------|------|
| `>` | 大于 |
| `<` | 小于 |
| `>=` | 大于等于 |
| `<=` | 小于等于 |
| `==` | 等于（比较，不是赋值） |
| `!=` | 不等于 |

- 比较运算结果一定是布尔值 `True` 或 `False`
- ⚠️ 注意：比较相等用 `==`，不要写成 `=`（那是赋值）

### 3. 逻辑运算符
| 运算符 | 说明 | 关键字 | 不要用 |
|--------|------|--------|--------|
| 逻辑与 | 两个都为真才是真 | `and` | `&`（位运算） |
| 逻辑或 | 只要一个为真就是真 | `or` | `|`（位运算） |
| 逻辑非 | 取反 | `not` | `!` |

**短路求值**：
- `A and B` - 如果A是False，B不会执行
- `A or B` - 如果A是True，B不会执行

### 4. 赋值运算符
- 基本：`a = 10`
- 复合赋值：
  ```python
  a += 5   # 等价于 a = a + 5
  a -= 5   # 等价于 a = a - 5
  a *= 5   # 等价于 a = a * 5
  a /= 5   # 等价于 a = a / 5
  ```

### 5. 运算符优先级（从高到低）
1. 幂运算 `**`
2. 正负号 `+x`, `-x`
3. 乘除 `*`, `/`, `//`, `%`
4. 加减 `+`, `-`
5. 比较 `>`, `<`, `>=`, `<=`, `==`, `!=`
6. 逻辑非 `not`
7. 逻辑与 `and`
8. 逻辑或 `or`

- 不确定就加括号 `()`，清晰优先

## 练习目标
- ✓ 掌握各种算术运算符的使用
- ✓ 正确使用比较运算符得到布尔结果
- ✓ 区分 `&/|` 和 `and/or`，逻辑运算用 `and/or`
- ✓ 能根据描述写出正确的条件表达式（比如闰年判断）
