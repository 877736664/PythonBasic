# 04-函数 - 学习要点

## 核心知识点

### 1. 为什么要用函数
- **代码复用**：相同逻辑只写一次
- **模块化**：把大问题拆成小函数，便于维护
- **方便测试**

### 2. 函数定义
```python
def 函数名(参数1, 参数2):
    """文档字符串（说明函数作用）"""
    函数体
    return 返回值  # 不需要返回可以省略return
```

- 定义用 `def` 关键字
- 函数名后必须有括号和冒号
- 缩进：函数体要缩进
- `return`：返回结果，结束函数

### 3. 函数调用
```python
结果 = 函数名(实参1, 实参2)
```

- 定义后不会自动执行，必须调用才会执行

### 4. 参数
- **形参**：定义函数时写的参数（占位符）
- **实参**：调用函数时传入的实际值

**默认参数：**
```python
def greet(name, message="你好"):
    print(f"{message}, {name}")

greet("张三")                # 使用默认值 message="你好"
greet("李四", "早上好")    # message使用传入的"早上好"
```
- 默认参数要放在参数列表最后

**可变参数：**
```python
def average(*nums):
    # *nums 把所有位置参数打包成元组
    for num in nums:
        ...
```
- `*args`（习惯命名）接收任意多个位置参数，类型是元组

### 5. 返回值
- 函数可以用 `return` 返回一个值
- 可以返回任何类型
- 如果没有 `return`，默认返回 `None`
- **返回多个值**：用逗号分隔，实际返回的是元组
  ```python
  def find_max_min(numbers):
      return max_val, min_val  # 返回元组，可以用多个变量接收

  max_v, min_v = find_max_min([1, 2, 3])
  ```

### 6. 变量作用域
- **局部变量**：函数内部定义的变量，只能在函数内部访问
- **全局变量**：函数外部定义的变量，函数内部可以读取，修改需要用 `global` 声明

### 7. 递归
- 函数自己调用自己
- 必须有**基线条件**（递归出口），否则会栈溢出
- 示例：阶乘、斐波那契

## 练习目标
- ✓ 能正确定义和调用函数
- ✓ 理解参数和返回值
- ✓ 掌握默认参数、可变参数
- ✓ 理解递归并能写出简单递归
- ✓ 知道变量作用域的概念
