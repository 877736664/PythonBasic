# 08-集合 - 学习要点

## 核心知识点

### 1. 什么是集合
- 集合是**无序、不重复**的可变容器
- 类似数学中的集合，支持交、并、差运算
- 用大括号 `{}` 表示，元素不重复

```python
# 创建集合
s1 = {1, 2, 3}
s2 = set()  # 空集合，不能用 {}（那是空字典）
# 去重：把列表转集合自动去重
lst = [1, 2, 2, 3]
s = set(lst)  # {1, 2, 3}
```

### 2. 基本操作

| 操作 | 写法 | 说明 |
|------|------|------|
| 添加 | `s.add(x)` | 添加一个元素 |
| 删除 | `s.remove(x)` / `s.discard(x)` | remove不存在会报错，discard不报错 |
| 长度 | `len(s)` |
| 判断存在 | `x in s` | 返回True/False |

### 3. 集合运算

| 运算 | 运算符 | 说明 |
|------|--------|------|
| 并集 | `a | b` | 两个集合所有元素 |
| 交集 | `a & b` | 同时在两个集合的元素 |
| 差集 | `a - b` | 在a不在b的元素 |
| 对称差 | `a ^ b` | 只在一个集合的元素 |
