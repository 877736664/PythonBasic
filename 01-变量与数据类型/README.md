# 01-变量与数据类型 - 学习要点

## 核心知识点

### 1. 变量定义
- Python 变量不需要声明类型，直接赋值即可
- 变量命名规则：字母、数字、下划线，不能以数字开头，不能是关键字
- 命名习惯：小驼峰 `myName` 或下划线命名 `my_name`（推荐Python用下划线）

### 2. 基本数据类型
| 类型 | 示例 | 说明 |
|------|------|------|
| `int` 整数 | `42`, `-10`, `0` | 整数，没有小数部分 |
| `float` 浮点数 | `3.14`, `-0.5`, `2.0` | 小数，注意浮点数精度问题 |
| `str` 字符串 | `"hello"`, 'world' | 文本，可用单引号或双引号 |
| `bool` 布尔值 | `True`, `False` | 只有两个值：真/假（首字母大写） |
| `NoneType` 空值 | `None` | 表示空，不是0也不是空字符串 |

### 3. 查看变量类型
- 使用 `type(变量)` 函数获取变量类型
```python
print(type(age))  # <class 'int'>
```

### 4. 类型转换
| 转换函数 | 作用 |
|----------|------|
| `int(x)` | 转为整数 |
| `float(x)` | 转为浮点数 |
| `str(x)` | 转为字符串 |
| `bool(x)` | 转为布尔值 |

### 5. 输入输出
- `print(内容)` - 输出到屏幕
- `input("提示文字")` - 从键盘读取输入，**返回值永远是字符串**

## 练习目标

- ✓ 理解各种基本数据类型的区别
- ✓ 掌握基本的类型转换
